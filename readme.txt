DXライブラリとOpenCVを使用しているので設定必要

DXライブラリ
1.http://homepage2.nifty.com/natupaji/DxLib/からDXライブラリをダウンロード
2.ダウンロードしたexeファイルを実行して展開
3.「ARnuigurumi_DXLib+CV.sln」と同じフォルダに「DX_Lib」フォルダを新規作成
4.展開したフォルダ内にある「プロジェクトに追加すべきファイル_VC用」フォルダを3で作成した「DX_Lib」フォルダにコピー

OpenCV
1.プロジェクトを開く
2.ソリューションエクスプローラの一番上、ソリューションを右クリック→「ソリューションのNuGetパッケージの管理」
3.検索で「opencv」を検索
4.「OpenCV」をインストール
